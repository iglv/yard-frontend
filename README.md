# yard-frontend
Complexes development demo web-site. 
Used: 
 - ReactJS
 - React Router
 - ESlint
 - Styled Components
 - Flexbox Grid

## Installation

Clone this repository. 
Install [Yarn](https://yarnpkg.com/) and run in terminal. 

```shell
yarn install
```

## Usage

Use for local server and demo. 
```shell
yarn start
```

Build for server. 
```shell
yarn build
```

## Deploy

For deploy use now.sh account.
